pycassa.connection – Low Level Connections¶

pycassa.connection.connect(keyspace, servers=None, framed_transport=True, timeout=None, credentials=None, retry_time=60, recycle=None, use_threadlocal=True)¶

Constructs a ConnectionPool. This is primarily available for reasons of backwards-compatibility; creating a ConnectionPool directly provides more options. All of the parameters here correspond directly with parameters of the same name in pycassa.pool.ConnectionPool.__init__()

pycassa.connection.connect_thread_local(*args, **kwargs)¶

Alias of connect()
